# 项目背景与上下文

## 项目简要说明

### 项目概述
AI面试助手是一个跨端应用，基于React、ANTD和zustand开发，旨在帮助用户回答面试问题。系统能够将用户和面试官的语音实时转换为文本，并利用AI模型提供回答和反馈。

### 项目目标
- 开发一个用户友好的面试助手应用
- 实现实时语音转文字功能
- 集成AI模型提供智能回答
- 支持多种AI模型选择
- 提供用户认证和数据持久化

### 核心功能

#### 音频转文字
- 全屏面试界面设计
- 前端获取麦克风音频流
- 实时音频流处理
- 将音频流发送给科大讯飞实时语音转写API
- 获取带时间戳的对话文字
- 流式显示在前端并转发给后端
- 支持角色分离显示
- 实现错误自动恢复

#### 回答AI
- 前端将文字+预置Prompt拼接
- 发送给基于deepseek-coder模型的API
- 获取AI回复
- 以打字机形式流式显示在前端并转发给后端
- 支持多模型切换
- 提供个性化回答
- 实现智能问题分析

## 产品上下文

### 目标用户群体
- **求职者**: 希望提升面试技巧的个人
- **面试教练**: 专业面试培训师
- **面试培训机构**: 提供面试培训服务的机构
- **技能提升者**: 对提升面试技巧感兴趣的个人

### 用户核心需求
- 希望在无压力环境下练习面试
- 获取即时反馈和改进建议
- 记录和回顾面试历史
- 选择不同类型的模型辅助练习
- 实时语音转写和对话显示
- 流畅的面试体验
- 可靠的错误处理机制

### 用户旅程

#### 新建面试会话
1. 用户注册/登录应用
2. 进入面试会话历史页面, 选择新建面试会话
3. 进入全屏面试会话页面
4. 点击"开始面试"按钮
5. 授权麦克风访问权限
6. 开始录音并回答面试问题
7. 实时查看语音转写结果
8. 获取AI实时反馈
9. 保存面试记录供后续回顾

#### 查看历史面试会话
1. 用户注册/登录应用
2. 进入面试会话历史页面, 选择查看历史会话
3. 进入历史会话页面, 查看历史会话记录
4. 可查看完整的对话内容
5. 可查看语音转写记录
6. 可查看AI反馈内容

#### 配置大模型API
1. 用户注册/登录应用
2. 进入面试会话历史页面, 在左侧Menu选择API配置
3. 在API配置页, 用户添加大模型供应商, baseUrl, APIKey等信息
4. 配置科大讯飞语音转写API参数
5. 测试API连接状态
6. 保存配置信息

## 技术上下文

### 前端技术栈
- **React + TypeScript**: 现代化前端开发框架
- **ANTD**: UI组件库，提供丰富的组件
- **Zustand**: 轻量级状态管理库
- **Axios**: HTTP请求库
- **Styled-components**: CSS-in-JS样式管理
- **Vite**: 快速构建工具
- **OpenAPI-typescript-codegen**: API代码生成工具
- **WebSocket**: 实时通信协议
- **WebAudio API**: 音频处理接口

### 后端技术栈
- **FastAPI**: 高性能Python Web框架
- **PostgreSQL**: 关系型数据库
- **JWT**: 用户认证机制
- **SQLAlchemy**: 对象关系映射(ORM)
- **Pydantic**: 数据验证库
- **Uvicorn**: ASGI服务器
- **WebSocket**: 实时通信支持
- **Redis**: 缓存数据库

### 外部API集成
